# Psych Analyzer (shared_lib)

Библиотека для валидации и психологического анализа результатов тестирования сотрудников. Модуль обработки данных в экосистеме проектов Mental Health.

##  Основные возможности

* **Валидация ответов:** Проверка входных массивов на логическую целостность (защита от случайных кликов).
* **Расчет индексов:** Автоматический подсчет баллов выгорания и психологического благополучия.
* **Интерпретация результатов:** Генерация текстовых статусов и персонализированных рекомендаций на основе набранных баллов.
